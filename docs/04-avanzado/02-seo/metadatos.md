# Configuración de Metadatos

Cada archivo Markdown en FusionDoc puede definir sus propios metadatos en la cabecera (frontmatter).

## Ejemplo de Frontmatter SEO
```yaml
---
title: Mi Título Optimizado
description: Una descripción cautivadora para los buscadores.
ogImage: /assets/hero.png
---
```

> [!TIP]
> Google recomienda mantener las descripciones entre 150 y 160 caracteres.
