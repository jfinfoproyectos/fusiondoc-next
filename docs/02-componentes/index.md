# Componentes

Explora nuestra biblioteca de componentes interactivos para mejorar tu documentación.

## Componentes Disponibles
- **CodeTabs**: Pestañas para mostrar código en múltiples versiones (npm, yarn, etc).
- **Terminal**: Una terminal interactiva para mostrar comandos.
- **Steps**: Guías paso a paso.
- **Accordion**: Contenido desplegable para organizar información.
- **Carousel**: Galerías de imágenes o bloques de contenido.
- **ZoomImage**: Imágenes con efecto de ampliación a pantalla completa.
- **Video/Audio**: Integración de contenido multimedia.
- **GithubRepo**: Tarjetas para publicar enlaces a repositorios de GitHub.
- **FileTree**: Árbol de archivos interactivo.
- **PropertyTable**: Tablas de API detalladas.
- **BentoGrid**: Cuadrículas de características modernas.
- **Timeline**: Línea de tiempo secuencial.
- **Badges**: Etiquetas de estado y categorías.
- **LivePreview**: Vista previa de código interactiva.
