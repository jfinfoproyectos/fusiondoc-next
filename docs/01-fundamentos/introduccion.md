# Bienvenido a FusionDoc

FusionDoc es una plataforma de documentación moderna diseñada para desarrolladores. Gracias a nuestra integración con **Next.js MDX**, puedes escribir documentación interactiva usando componentes de React directamente en tus archivos Markdown.

## Instalación Rápida

Para comenzar con FusionDoc, puedes ejecutar el siguiente comando en tu terminal:

<Terminal 
  title="Instalación"
  shell="bash"
  commands="npm install fusiondoc@latest\nnpm run init"
  staticText="Instalando dependencias de FusionDoc..."
/>

## ¿Por qué usar FusionDoc?

- **Rendimiento**: Generación de sitios estáticos ultrarrápidos.
- **Componentes**: Biblioteca de componentes interactivos listos para usar.
- **Comunidad**: Gran ecosistema de temas y plugins.
